## 📋 项目概述

一个可视化的 Minecraft 命令方块和数据包命令编辑工具，支持：
- 命令可视化编辑和预览
- 命令方块链管理
- 数据包/函数文件生成
- NBT 数据编辑

## 🏗️ 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                    React Application                      │
├─────────────┬─────────────┬─────────────┬────────────────┤
│   命令编辑器  │   命令预览   │   NBT编辑器  │   导出模块     │
├─────────────┴─────────────┴─────────────┴────────────────┤
│                    Core Engine Layer                      │
├─────────────┬─────────────┬─────────────┬────────────────┤
│ Command     │ NBT         │ Datapack    │ Command        │
│ Parser      │ Parser      │ Generator   │ Validator     │
└─────────────┴─────────────┴─────────────┴────────────────┘
```

### 技术选型

| 层级 | 技术 | 用途 |
|------|------|------|
| 前端框架 | React 18 + TypeScript | UI 组件和状态管理 |
| 构建工具 | Vite 5 | 快速开发和构建 |
| 状态管理 | Zustand | 轻量级全局状态 |
| 样式方案 | Tailwind CSS | 原子化 CSS |
| UI 组件 | shadcn/ui | 可定制组件库 |
| 图标 | Lucide React | 图标库 |
| NBT 解析 | prismarine-nbt | NBT 文件解析 |
| 拖拽 | dnd-kit | 命令方块拖拽排序 |

## 📝 技术笔记

### 命令解析器设计参考

参考 Mojang Brigadier 架构:
- [JSPrismarine/brigadier](https://github.com/JSPrismarine/brigadier) - JavaScript 实现
- [Mojang/brigadier](https://github.com/Mojang/brigadier) - 原始 Java 实现

### NBT 解析库选择

使用 [prismarine-nbt](https://github.com/PrismarineJS/prismarine-nbt):
- 支持 Node.js 和浏览器
- 支持 big-endian/little-endian/varint
- 活跃维护

### Minecraft 命令参考

- [Minecraft Wiki - Commands](https://minecraft.wiki/w/Commands)
- [Bedrock Command Documentation](https://wiki.bedrock.dev/)
